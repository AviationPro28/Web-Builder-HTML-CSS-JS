This is a minimal index.html to get your GitHub Pages site working. Customize the text, add assets, or drop in your app build.

View repo
Tip: put this file at the repo root or in a /docs folder and enable Pages in repo settings.

Quick checklist

  • Commit this file to the default branch (main/master) or gh-pages.
  • If you use /docs, select the branch + /docs folder in Pages settings.
  • Set repo name to username.github.io for a root user site (optional).

Publish commands

git add index.html
git commit -m "Add index.html"
git push origin main
